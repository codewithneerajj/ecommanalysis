# 🛒 Ecomm Project — Data Analysis

## 📌 Project Overview
This project analyzes e-commerce data using SQL, Power BI, and Excel.

## 📁 Project Structure
- **SQL/** — MySQL queries for data extraction and analysis
- **PowerBI/** — Interactive dashboards (.pbix file)
- **Excel/** — Data sheets and pivot tables

## 🛠️ Tools Used
- MySQL Workbench
- Microsoft Power BI
- Microsoft Excel

## 📊 How to Use
1. Run the SQL file in MySQL Workbench
2. Open the .pbix file in Power BI Desktop
3. Open the Excel file in Microsoft Excel
